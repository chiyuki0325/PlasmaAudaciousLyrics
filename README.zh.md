[English](/README.md)

Lyrica 是一个简洁轻量的 Linux 桌面歌词显示软件。  
不同于类似的项目，比如 [Waylyrics](https://github.com/waylyrics/waylyrics)，Lyrica 通过桌面挂件 / 拓展与桌面环境集成，将歌词直接显示在面板上。  
目前项目正在开发中，但目前的版本已经可以使用。  

项目的名字来源于东方 Project 的角色莉莉卡·普莉兹姆利巴。
